# SULTANOVA AI — Описание проекта

> Последнее обновление: март 2026

---

## Что это за проект

Персональный сайт-портфолио и лендинг для личного бренда **Альмиры Султановой** под маркой **SULTANOVA AI**.

Цель сайта — привлекать клиентов и конвертировать их в заявку через Telegram.
Позиционирование: не фрилансер, а **архитектор AI-решений и цифровых продуктов**.

Технически: одностраничный HTML-файл без фреймворков и зависимостей.
Хостинг: GitHub Pages → `https://almirasultabova.github.io/my-first-project/`

---

## Для кого

### Целевая аудитория

**Эксперты и предприниматели**, которые хотят:
- Сайт или лендинг под продукт / запуск
- Telegram-бота для заявок, продаж или клиентского сервиса
- AI-ассистента на своих данных
- Подписочный продукт в Telegram
- Квиз, калькулятор, Mini App
- Автоматизацию рутинных процессов
- MVP нового digital-сервиса

Три сегмента:
1. **Малый и средний бизнес** — нужен инструмент под ключ, важна надёжность
2. **Эксперты** (коучи, юристы, онлайн-школы) — лендинг, личный бренд, AI-ассистент
3. **Бизнес под автоматизацию** — внедрить AI, не понимают с чего начать

7. **Мой подход** — принципы работы, «система под задачу»
8. **Процесс** — 4 шага от разбора задачи до внедрения
9. **Сильный блок** — цитата + 5 типов задач + CTA
10. **Отзывы** — 3 карточки + кнопка на Telegram-канал
11. **FAQ** — 6 вопросов и ответов
12. **Контакты** — прямая ссылка в Telegram + форма (открывает TG с предзаполненным сообщением)
13. **Footer** — логотип, навигация, соцсети

## Страницы, которые планируются

### Ближайшие (по мере появления реальных проектов)

- **Портфолио / кейсы** — отдельная страница с реальными проектами.
  Каждый кейс: клиент, задача, решение, результат, скриншоты.
  Сейчас в `index.html` — абстрактные сценарии, заменятся на реальные.

- **Статистика в «Обо мне»** — блок с цифрами (проектов, клиентов, стран) убран.
  Вернётся, когда будут честные данные.

### В перспективе (переход к AI-студии)

- **Страница студии** — позиционирование SULTANOVA AI как студии,
  не просто личного специалиста. С лестницей продуктов и форматами сотрудничества.
- **Блог / заметки** — интеграция или отдельная страница с материалами из Telegram-канала.
- **Страница отдельного продукта** — лендинг под конкретный AI-инструмент или сервис,
  когда появится собственный продукт.

Подробнее — в [vision.md](vision.md).

## Стиль дизайна

**Тема:** тёмная (dark mode), почти чёрный фон.

**Общий стиль:** современный tech / AI-эстетика.
Glassmorphism + gradient glow + noise texture.

### Цветовая палитра

| Переменная | Цвет | Роль |
|---|---|---|
| `--bg` | `#0A0A0F` | Основной фон |
| `--bg2` | `#0D1117` | Карточки, секции |
| `--bg3` | `#111827` | Вложенные элементы |
| `--p` | `#4f46e5` | Основной акцент (индиго) |
| `--ph` | `#6366f1` | Hover для основного |
| `--a` | `#06b6d4` | Вторичный акцент (cyan) |
| `--txt` | `#F0F4FF` | Основной текст |
| `--muted` | `#8892A4` | Второстепенный текст |
| `--grad` | indigo → cyan | Градиент кнопок, линий |
| `--grad-text` | white → lavender → cyan | Градиент заголовков |

### Типографика

- Шрифт: `Inter` (system stack как fallback)
- Заголовки секций: `clamp(1.9rem, 5vw, 3rem)`, weight 800
- Hero H1: `clamp(1.85rem, 4.5vw, 3.6rem)`, weight 800
- Текст: 0.88rem – 1.05rem, line-height 1.65–1.9

### UI-элементы

- Скруглённые карточки (`border-radius: 16px`)
- Тонкие границы `rgba(255,255,255,0.07)`
- Свечение на hover (`box-shadow` glow в цвет акцента)
- Анимация появления при скролле (IntersectionObserver, `.rv` + `.on`)
- Canvas-частицы в Hero с соединяющими линиями
- Кастомный курсор с инерцией (только desktop)
- Прогресс-бар скролла сверху страницы

### Адаптивность

- Mobile-first
- Брейкпоинты: 480px / 768px / 900px
- Сетка услуг: 4 колонки → 2 → 1
- Bento-сетка: 4 колонки → 2 → 1
