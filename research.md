### 1.2 Что цепляет в топовых портфолио (паттерны)

**Первый экран:**
- Большой заголовок с градиентом или анимацией (не «Welcome to my portfolio»)
- Интерактивный фон: WebGL, canvas-частицы, 3D-объект
- Сразу видно специализацию — без скролла
- Кастомный курсор

**Обязательные секции (встречаются в 90%+ портфолио):**
1. Hero — имя + специализация + CTA
2. Выбранные работы / кейсы (3–6 штук)
3. О себе — короткий текст + стек
4. Контакты

**Необязательные, но конвертирующие:**
- Логотипы клиентов
- Отзывы
- Блог / статьи
- «Доступен для работы» — статус

**Визуальные стили 2025:**
- Dark mode (#0a0a0a, #080808) — лидирует
- Монохром (чёрно-белый) + один акцент
- Glassmorphism карточки
- Крупная типографика как главный визуал

## 2. СТРУКТУРА ЛУЧШЕГО ЛЕНДИНГА

### Идеальная структура: 9 блоков

```
┌─────────────────────────────────────────┐
│  STICKY HEADER                           │
│  Имя/Лого · Навигация · CTA-кнопка      │
├─────────────────────────────────────────┤
│  1. HERO                                 │
│  Заголовок · Подзаголовок · 2 кнопки    │
├─────────────────────────────────────────┤
│  2. БОЛИ КЛИЕНТА                         │
│  «Знакомо?» · 4-5 болей · переход       │
├─────────────────────────────────────────┤
│  3. О СЕБЕ                               │
│  Фото · История · Стек · Статистика     │
├─────────────────────────────────────────┤
│  4. УСЛУГИ (3 карточки)                  │
│  Сайты · Приложения · ИИ-агенты         │
├─────────────────────────────────────────┤
│  5. ПОРТФОЛИО / КЕЙСЫ                    │
│  4-6 проектов с результатами             │
│                          [CTA — здесь]  │
├─────────────────────────────────────────┤
│  6. ПРОЦЕСС РАБОТЫ                       │
│  5 шагов · Сроки · Предсказуемость      │
├─────────────────────────────────────────┤
│  7. ОТЗЫВЫ                               │
│  3-5 с фото + логотипы клиентов         │
├─────────────────────────────────────────┤
│  8. FAQ                                  │
│  6-8 вопросов · Снятие возражений       │
├─────────────────────────────────────────┤
│  9. ФИНАЛЬНЫЙ CTA + КОНТАКТЫ             │
│  Форма · Telegram · Обещание ответа     │
├─────────────────────────────────────────┤
│  FOOTER                                  │
└─────────────────────────────────────────┘
```

### Детализация каждого блока

#### Блок 1 — HERO
**Цель:** захватить внимание за 5 секунд, передать ценность.

```
Статус:  ● Открыт для проектов с апреля 2026
Имя:     [ИМЯ ФАМИЛИЯ]
H1:      Строю цифровые продукты.
         Сайты. Приложения. ИИ-агенты.
Sub:     Помогаю бизнесу расти быстрее —
         через технологии, которые работают без вас.

[Обсудить проект]  [Смотреть работы ↓]
```

**Правило:** Первичный CTA — конкретный. Вторичный — мягкий (якорная ссылка).

#### Блок 2 — БОЛИ КЛИЕНТА
**Цель:** эмпатия + признание проблемы.

Примеры болей:
- «Наняли разработчика — получили сырой продукт и тишину в мессенджере»
- «Сайт есть, но заявок нет — непонятно почему»
- «Хотели внедрить ИИ, но решение оказалось слишком сложным для команды»
- «Три месяца разработки — дедлайн снова сорван»

Завершение: *«Именно для этого я здесь.»*

> По данным CopyHackers: страницы с блоком «боли клиента» конвертируют на 30–45% лучше.

#### Блок 3 — О СЕБЕ
Формула:
> «[Имя] — [роль] с [N лет] опытом. Начинал с [контекст]. Сейчас специализируюсь на [ниша], потому что [личная причина]. За это время [достижение в цифрах]. Верю, что [ценность].»

Обязательно: **реальное фото** (+35% доверия по Nielsen Norman Group).

#### Блок 4 — УСЛУГИ
3 карточки, не больше (закон Хика: больше выбора = меньше решений).

| Услуга | Описание | Срок |
|--------|----------|------|
| 🌐 Сайты и лендинги | Конверсионные сайты, которые приводят клиентов. Дизайн + код + аналитика. | от 2 нед. |
| 📱 Веб-приложения | SaaS, CRM, личные кабинеты, внутренние инструменты. Полный цикл. | от 4 нед. |
| 🤖 ИИ-агенты | Чатботы на ваших данных, автоворонки, GPT-ассистенты. Автоматизация рутины. | от 1 нед. |

#### Блок 5 — КЕЙСЫ
Формула кейса (STAR):
```
[Скриншот + ссылка на живой проект]

ЗАДАЧА: Что было у клиента, в чём боль
ЧТО СДЕЛАЛ: Конкретные действия
РЕЗУЛЬТАТ: Цифры до/после

[Смотреть проект →]
```

❌ Слабо: «Лендинг для компании X. React, Tailwind.»
✅ Сильно: «Конверсия выросла с 0.8% до 2.6% за первый месяц. +40 заявок ежемесячно.»

#### Блок 6 — ПРОЦЕСС
5 шагов, устраняют страх «а как это вообще работает»:
1. **Discovery-звонок** (бесплатно, 30 мин) — обсуждаем задачу
2. **Бриф и предложение** (1–2 дня) — план, сроки, стоимость
3. **Дизайн и прототип** (1–2 нед.) — макеты в Figma, согласование
4. **Разработка** (3–8 нед.) — демо каждые 2 недели
5. **Запуск и поддержка** — 30 дней сопровождения после

#### Блок 7 — ОТЗЫВЫ
Формула сильного отзыва:
> «Было [проблема]. [Имя специалиста] [что сделал]. Результат — [конкретная цифра].»
> — **Имя Фамилия**, Должность, Компания + фото

> 92% покупателей читают отзывы перед решением. Отзыв с фото работает в 7× эффективнее анонимного (BrightLocal).

#### Блок 8 — FAQ
Обязательные вопросы:
- Сколько стоит? (хотя бы диапазон)
- Сколько длится разработка?
- Что если мне не понравится результат?
- Можно ли внедрить ИИ в уже существующий сайт?
- Работаете ли вы с нерезидентами?

#### Блок 9 — ФИНАЛЬНЫЙ CTA
```
Заголовок: "Расскажите о задаче — предложу решение бесплатно"
Форма:     Имя + Telegram/Email + Краткое описание
Telegram:  [Написать в Telegram]
Обещание:  Отвечаю в течение 2 часов в рабочие дни
```

### Типичные ошибки

| Ошибка | Что делать |
|--------|-----------|
| Нет цифр в кейсах | Добавить метрики до/после |
| Больше 3 услуг | Сократить до 3 основных |
| Нет фото автора | Сделать профессиональное фото |
| CTA только в конце | CTA в hero + середине + финале |
| Форма с 7+ полями | Только: Имя + контакт + задача |
| Нет мобильной версии | Mobile-first разработка |

## 3. ТРЕНДЫ И ФИШКИ 2025–2026

### Топ-12 трендов с реализацией

#### Тренд 1 — Scroll-Triggered Animations ⭐⭐⭐
Элементы появляются/трансформируются при прокрутке. Удерживает внимание.

```css
.reveal {
  opacity: 0;
  transform: translateY(60px);
  transition: opacity 0.8s cubic-bezier(0.16, 1, 0.3, 1),
              transform 0.8s cubic-bezier(0.16, 1, 0.3, 1);
}
.reveal.visible { opacity: 1; transform: translateY(0); }
```
```js
const observer = new IntersectionObserver(entries => {
  entries.forEach(e => { if (e.isIntersecting) e.target.classList.add('visible'); });
}, { threshold: 0.15 });
document.querySelectorAll('.reveal').forEach(el => observer.observe(el));
```
**Применение:** карточки проектов, секция услуг, статистика.

#### Тренд 2 — Glassmorphism 2.0 ⭐⭐
Матовое стекло с реальными бликами и многослойным blur.

```css
.glass-card {
  background: rgba(255, 255, 255, 0.07);
  backdrop-filter: blur(24px) saturate(160%);
  border: 1px solid rgba(255, 255, 255, 0.12);
  border-radius: 20px;
  box-shadow: 0 8px 32px rgba(0,0,0,0.3),
              inset 0 1px 0 rgba(255,255,255,0.2);
}
```
**Применение:** карточки навыков, sticky nav, форма контактов.

#### Тренд 3 — Kinetic Typography (кинетическая типографика) ⭐⭐⭐
Текст как главный визуал hero без фотографий.

```css
.hero-name {
  font-size: clamp(3rem, 10vw, 9rem);
  font-weight: 900;
  letter-spacing: -0.03em;
  clip-path: inset(0 100% 0 0);
  animation: reveal-word 1s cubic-bezier(0.77, 0, 0.175, 1) forwards;
}
@keyframes reveal-word { to { clip-path: inset(0 0% 0 0); } }
```
**Применение:** имя + должность в hero.

#### Тренд 4 — Custom Cursor с магнитным эффектом ⭐⭐
Маркер «дорогого» сайта. Курсор с lerp-инерцией.

```css
.cursor {
  width: 8px; height: 8px;
  background: #fff; border-radius: 50%;
  position: fixed; pointer-events: none;
  mix-blend-mode: difference; /* инвертирует цвет под курсором */
}
.cursor-follower {
  width: 40px; height: 40px;
  border: 1px solid rgba(255,255,255,0.5);
  border-radius: 50%;
  position: fixed; pointer-events: none;
  transition: width 0.3s, height 0.3s;
}
```
```js
let mouseX = 0, mouseY = 0, fX = 0, fY = 0;
document.addEventListener('mousemove', e => { mouseX = e.clientX; mouseY = e.clientY; });
(function animate() {
  fX += (mouseX - fX) * 0.1; fY += (mouseY - fY) * 0.1;
  follower.style.left = fX + 'px'; follower.style.top = fY + 'px';
  requestAnimationFrame(animate);
})();
```
**Применение:** глобальный эффект, особенно на dark-теме.

#### Тренд 6 — CSS Scroll-Driven Animations (нативный API) ⭐⭐⭐
Анимации привязаны к скроллу без JS. Chrome 115+.

```css
/* Прогресс-бар без JS */
.scroll-progress {
  position: fixed; top: 0; left: 0;
  height: 3px; background: var(--accent);
  transform-origin: left;
  animation: progress linear;
  animation-timeline: scroll(root);
}
@keyframes progress { from { transform: scaleX(0); } to { transform: scaleX(1); } }

/* Элементы появляются при скролле */
.section-item {
  animation: fade-up linear both;
  animation-timeline: view();
  animation-range: entry 0% entry 40%;
}
@keyframes fade-up {
  from { opacity: 0; transform: translateY(60px); }
  to   { opacity: 1; transform: translateY(0); }
}
```
**Применение:** прогресс-бар, появление секций, параллакс.

#### Тренд 7 — AI-эстетика (визуальный язык ИИ) ⭐⭐⭐
Пульсирующие сетки, gradient orbs, scanning-линии.

```css

  transform: perspective(500px) rotateX(45deg);

/* Gradient orb — пульсирующий шар */
.ai-orb {

@keyframes orb-pulse {
  from { transform: scale(1); opacity: 0.5; }

/* Scanning line */
```

#### Тренд 9 — Horizontal Scroll секции ⭐⭐
Галерея проектов горизонтальным слайдером через вертикальный скролл.

```css
.h-wrapper { height: 400vh; position: relative; }
.h-sticky  { position: sticky; top: 0; height: 100vh; overflow: hidden; }
.h-track   { display: flex; gap: 2rem; width: max-content; }
.h-item    { width: 80vw; height: 80vh; flex-shrink: 0; border-radius: 20px; }
```
```js
window.addEventListener('scroll', () => {
  const progress = Math.max(0, Math.min(1,
    -wrapper.getBoundingClientRect().top / (wrapper.offsetHeight - window.innerHeight)
  ));
  track.style.transform = `translateX(${-progress * (track.scrollWidth - window.innerWidth)}px)`;
});
```

#### Тренд 12 — Micro-interactions ⭐⭐
Физичность каждого действия — кнопка нажимается, поле оживает.

```css

}
.btn:active {
  transform: translateY(4px);
  box-shadow: 0 2px 0 rgba(0,0,0,0.3);
}

/* Input с анимированным border */
.input { border-bottom: 2px solid #333; position: relative; }
.input::after {
  content: ''; position: absolute; bottom: -2px; left: 0;
  width: 0; height: 2px; background: #6366f1;
  transition: width 0.3s;
}
.input:focus::after { width: 100%; }
```

```css
  --teal:    #14B8A6;
}
```

- **Код/акцент:** JetBrains Mono

## 7. ЭКСПЕРТНАЯ ОЦЕНКА ИДЕИ

> Проект: лендинг-портфолио цифрового мастера — сайты, приложения, ИИ-агенты.
> Три независимые точки зрения: UX, маркетинг, клиент.

---

### 🎨 ЭКСПЕРТ 1 — UX-дизайнер (10 лет опыта)

#### Типичные ошибки новичков в структуре

**1. Нет ясности на первом экране**
Посетитель смотрит на hero 5 секунд. Если непонятно «кто ты, чем занимаешься, чем можешь помочь» — он уходит. Ошибка: «Welcome to my portfolio» или абстрактный слоган без конкретики.

**2. Перегруз навигации**
Пункты меню: Home · About · Skills · Portfolio · Services · Blog · Contact · Hire Me — пользователь теряется. Правило: не больше 4-5 пунктов, лучше 3.

**3. Портфолио без контекста**
Скриншоты без объяснения: что за проект, для кого, какую задачу решал, какой результат. Кейс без результата — это просто картинка.

**4. Форма контактов в самом конце и только там**
CTA должен быть минимум в 3 местах: hero → середина (после кейсов) → финал.

**5. Нет визуальной иерархии**
Все элементы одного размера и веса — взгляд не знает куда идти. Нужны: крупный заголовок → подзаголовок → контент → CTA.

**6. Анимации ради анимаций**
Parallax на каждом блоке, fade-in у каждого слова, бесконечные loop-анимации фона — это утомляет и замедляет загрузку. Анимация должна помогать вести взгляд, а не развлекать.

**7. Отсутствует мобильная версия**
40-60% клиентов зайдут с телефона. Бюджетный horizontal-scroll кейсов или таблица навыков на мобиле — это катастрофа.

#### Что ОБЯЗАТЕЛЬНО должно быть

| Элемент | Зачем |
|---------|-------|
| Sticky nav с CTA-кнопкой | Кнопка «Написать» всегда под рукой |
| Статус доступности | «Открыт для проектов» снимает барьер первого контакта |
| Реальное фото | Доверие. Люди нанимают людей, не логотипы |
| Live-ссылки на проекты | Подтверждение, что проекты реальные |
| Якорные ссылки | Плавный скролл к нужному разделу |
| Время ответа | «Отвечаю в течение 2 часов» — снижает тревогу клиента |
| Микро-интеракции | Hover на кнопках, состояния полей формы — ощущение качества |

#### Что раздражает и заставляет уйти

- **Долгая загрузка** (>3 сек на мобиле) — 53% уходят (данные Google)
- **Автовоспроизведение видео со звуком** — закрывают мгновенно
- **Попап «подпишись» на первой секунде** — раздражение без доверия
- **Курсив мелким шрифтом** — нечитаемо на мобиле
- **Стоковые фото** вместо реальных работ — ощущение обмана
- **Форма с 6+ полями** — «зачем мне вводить ИНН чтобы получить консультацию?»
- **Нет цен даже в виде диапазона** — подозрение, что дорого или мутно

---

### 📊 ЭКСПЕРТ 2 — Маркетолог, специалист по конверсии

#### Оффер, который зацепит клиента

Слабый оффер: *«Разрабатываю сайты и приложения»*
Всё. Так говорят все. Это описание деятельности, а не ценности.

**Формула сильного оффера:**
> [Результат для клиента] + [за какое время] + [без чего] + [гарантия]

Примеры для цифрового мастера:

| Услуга | Слабо | Сильно |
|--------|-------|--------|
| Сайт | «Создаю сайты» | «Лендинг, который начнёт приносить заявки за 14 дней» |
| Приложение | «Разрабатываю веб-приложения» | «MVP за 4 недели — от идеи до первых пользователей» |
| ИИ-агент | «Делаю AI-ботов» | «ИИ-агент, который закрывает типовые запросы вместо менеджера. Экономит 20+ часов в неделю» |

**Главный оффер для hero:**
> Строю цифровые инструменты, которые работают, пока вы спите.
> Сайт. Приложение. ИИ-агент — под ключ.

#### Триггеры доверия (по убыванию важности)

1. **Кейсы с цифрами** — «+40 заявок в месяц», «время обработки заявок сократилось с 4 часов до 12 минут»
2. **Логотипы реальных клиентов** — даже 3-4 узнаваемых лого дают эффект
3. **Отзывы с фото и должностью** — «Директор по маркетингу, ООО X»
4. **Цифры опыта** — количество проектов, лет, клиентов из разных стран
5. **Живые ссылки** на работающие проекты (не скриншоты)
6. **Гарантия** — «Правки в течение 30 дней после запуска»
7. **Процесс работы** — предсказуемость снижает страх «а вдруг пропадёт с деньгами»
8. **Публичность** — GitHub, Telegram-канал, статьи — признак эксперта, а не анонима

#### Путь посетителя: от «зашёл» до «оставил заявку»

```
ОСОЗНАНИЕ (0–5 сек)
Зашёл с рекламы / сарафана / поиска
↓ Вопрос: «Это мне подходит?»
Hero → ясный заголовок + специализация

ИНТЕРЕС (5–30 сек)
Читает hero, видит кнопки
↓ Вопрос: «Он реально умеет?»
Листает к кейсам → видит результаты

ОЦЕНКА (30 сек – 3 мин)
Читает 1-2 кейса, отзывы, смотрит «Обо мне»
↓ Вопрос: «Можно ли ему доверять?»
Фото + отзывы + логотипы клиентов

НАМЕРЕНИЕ
Думает: «Хочу, но не знаю сколько стоит»
↓ Смотрит FAQ → находит диапазон цен / условия

ДЕЙСТВИЕ
CTA в нужный момент: «Обсудить проект»
↓
Форма: имя + контакт + краткое описание → ОТПРАВИТЬ
```

**Ключевой инсайт:** 70% людей, которые ушли без заявки — хотели, но что-то остановило. Чаще всего: нет цен, нет гарантий, нет понятного следующего шага.

#### Фишки конверсии

- **Счётчик «2 слота в апреле»** — дефицит = срочность
- **Бесплатный первый шаг** — «Discovery-звонок 30 мин бесплатно» снимает барьер
- **Конкретное обещание** — «Отвечу в течение 2 часов в рабочее время»
- **Telegram-кнопка** рядом с формой — часть людей не хочет заполнять форму

### 👤 ЭКСПЕРТ 3 — Потенциальный клиент (владелец бизнеса, 35 лет)

*«Я открываю очередного фрилансера. У меня было уже двое — один пропал с предоплатой, другой сдал через 3 месяца вместо 3 недель. Я устал. У меня 5 минут.»*

#### Что он смотрит в первые 5 секунд

1. **Кто ты?** — имя, фото (или нет фото — уже минус)
2. **Что делаешь?** — сайты / приложения / ИИ — ок, понятно
3. **Это вообще реально?** — быстро скроллит к работам
4. **Где отзывы?** — ищет глазами, нет — подозрение

**Что заставляет остаться:**
- Вижу реальные проекты с живыми ссылками — «это работает»
- Вижу цифры результатов — «это не просто красиво»
- Вижу фото нормального человека, не стоковое — «он настоящий»
- Процесс работы расписан — «понятно, что будет происходить»
- Есть цены или хотя бы диапазон — «не надо угадывать»

**Что заставляет закрыть вкладку:**

| Триггер ухода | Мысль клиента |
|---------------|---------------|
| Нет фото | «Кто этот человек вообще?» |
| Только скриншоты без ссылок | «Может это чужие работы» |
| Нет цен вообще | «Значит дорого или будет торговаться» |
| Форма с 7+ полями | «Мне лень, напишу другому» |
| Нет телефона / мессенджера | «Как вообще связаться?» |
| Сайт грузится дольше 3 сек | Закрыл, не подождал |
| Нет отзывов с реальными именами | «Сам придумал» |
| Написано «более 5 лет опыта» без доказательств | «Все так пишут» |

**Главный вопрос клиента, который он не задаст вслух:**
> *«Если я дам тебе деньги — ты не пропадёшь и сделаешь нормально?»*

Всё содержание лендинга должно отвечать на этот вопрос.

Собранные выводы от всех трёх экспертов:

#### Обязательно сделать ✅

- [ ] **Hero с конкретикой** — специализация + кому помогаешь + результат, не «добро пожаловать»
- [ ] **Реальное фото** — доверие выше на 35% (Nielsen Norman)
- [ ] **Кейсы с цифрами** — минимум 3, каждый с результатом до/после
- [ ] **Живые ссылки** на проекты — не просто скриншоты
- [ ] **3 CTA по странице** — hero + после кейсов + финал
- [ ] **Диапазон цен или FAQ о стоимости** — «от X₽» снимает страх неизвестности
- [ ] **Процесс работы** — 4-5 шагов, что происходит после заявки
- [ ] **Отзывы с именем + должность + фото** — минимум 3
- [ ] **Статус доступности** — «Открыт / занят до апреля»
- [ ] **Telegram-кнопка** рядом с формой
- [ ] **Мобильная версия** — тестировать на реальном телефоне
- [ ] **Загрузка <3 сек** — сжать изображения, убрать лишние скрипты

#### Хорошо добавить ⚡

- [ ] Логотипы клиентов (даже 3-4 штуки)
- [ ] Гарантия (30 дней правок после запуска)
- [ ] Счётчик свободных слотов («2 места в апреле»)
- [ ] Бесплатный первый шаг («Discovery-звонок 30 мин»)
- [ ] Open Graph для красивого превью в мессенджерах

#### Не делать ❌

- [ ] Автовоспроизведение видео со звуком
- [ ] Попап при входе
- [ ] Форма с более чем 3-4 полями
- [ ] «Welcome to my portfolio» в заголовке
- [ ] Навигация с 7+ пунктами
- [ ] Анимации на каждом элементе (только смысловые)
- [ ] Стоковые фото вместо реальных работ
