## 1. ЦЕЛЬ И АУДИТОРИЯ

### Главная цель сайта
Конвертировать посетителя в **заявку на проект** через Telegram или форму.
Вторичная цель — сформировать доверие и позиционировать автора как эксперта, а не просто исполнителя.

### Целевая аудитория (три сегмента)

#### Сегмент A — Малый и средний бизнес
- Владелец / директор, 30–50 лет
- Хочет сайт / приложение под ключ
- Боль: прошлые подрядчики пропадали или затягивали
- Ищет: надёжность, процесс, результат в цифрах
- Решение: кейсы с результатами + чёткий процесс работы + гарантия

#### Сегмент B — Эксперты и фрилансеры
- Коуч, психолог, юрист, онлайн-школа, 25–45 лет
- Нужен лендинг или личный бренд-сайт
- Боль: дорого в агентствах, непонятно что получат
- Решение: примеры схожих проектов + фиксированная стоимость + быстрые сроки

#### Сегмент C — Компании под автоматизацию
- Руководитель или CTO, хочет внедрить ИИ
- Боль: не понимает с чего начать, боится сложности
- Решение: простое объяснение ИИ-агентов + конкретные кейсы экономии времени

### Главный вопрос клиента (из экспертной оценки)
> *«Если я дам тебе деньги — ты не пропадёшь и сделаешь нормально?»*

Каждый блок лендинга отвечает на этот вопрос.

### STICKY HEADER

**Поведение:** фиксируется при скролле вниз, появляется с blur-фоном после 80px от верха.

**Левая часть — логотип/имя:**
```
[ИМЯ ФАМИЛИЯ]   или   [ИФ]  (монограмма)
```

**Навигация (центр или справа, 4 пункта):**
```
Услуги · Кейсы · Обо мне · Контакты
```

**Правая часть:**
```
● Открыт для проектов    [Обсудить проект]
```
- Зелёная точка (анимированный pulse) + текст статуса
- Кнопка «Обсудить проект» — ведёт к секции #9 (форма)

### СЕКЦИЯ 1 — HERO

**Цель:** захватить внимание за 5 секунд, передать специализацию, спровоцировать скролл.

**Фон:** canvas-анимация (частицы + соединительные линии) + gradient orb + noise texture поверх.

**Верхний бейдж:**
```
⚡ Открыт для новых проектов · апрель 2026
```
Горизонтальный чип с пульсирующей точкой.

**Главный заголовок (H1, ~9vw):**
```
Строю цифровые
инструменты.
```
Градиент: белый → indigo → cyan.

**Подзаголовок (специализация):**
```
Сайты · Приложения · ИИ-агенты
```
Анимация: каждое слово появляется последовательно (stagger).

**Описание (max-width 560px):**
```
Помогаю бизнесу расти через технологии,
которые работают без вас.
```

**Два CTA:**
```
[ Обсудить проект ]     [ Смотреть работы ↓ ]
  (primary, pulse)          (secondary, якорь)
```

**Микростатистика под кнопками:**
```
15+ проектов   ·   3 направления   ·   ответ за 2 часа
```
Три небольших числа в ряд, разделённые точками.

**Скролл-индикатор внизу:**
```
↓  прокрутите
```
Анимация bounce.

```css
  --primary:    #4f46e5;   /* Indigo — кнопки, акценты */
  --primary-h:  #6366f1;   /* Hover состояние */
  --accent:     #06b6d4;   /* Cyan — ИИ-направление, теги */
  --accent-h:   #22d3ee;

  /* Текст */
  --text:       #F0F4FF;   /* Основной текст */
  --text-muted: #8892A4;   /* Второстепенный текст */

  /* Границы и эффекты */
  --border:     rgba(255,255,255,0.08);
  --glow-p:     0 0 40px rgba(79,70,229,0.35);
  --glow-a:     0 0 40px rgba(6,182,212,0.25);

  /* Градиенты */
  --grad-main:  linear-gradient(135deg, #4f46e5, #06b6d4);
  --grad-text:  linear-gradient(135deg, #ffffff 20%, #c7d2fe 60%, #22d3ee 100%);
}
```

**Цветовая логика:**
- `--primary` (indigo) — основные действия: кнопки, заголовки
- `--accent` (cyan) — ИИ-направление, лейблы, теги, иконки ИИ-агентов
- Тёмный фон — создаёт ощущение глубины и экспертности
- Градиент primary→accent — связывает два направления (разработка + ИИ)

### Типографика

**Шрифт:** Inter Variable (загружается локально)

```css
@font-face {
  font-family: 'Inter';
  src: url('inter-variable.woff2') format('woff2-variations');
  font-weight: 100 900;
  font-style: normal;
  font-display: swap;
}
```

**Шкала размеров:**

| Элемент | Размер | Вес | Letter-spacing |
|---------|--------|-----|----------------|
| H1 (hero) | clamp(3.5rem, 10vw, 8rem) | 800 | -0.04em |
| H2 (секции) | clamp(2rem, 5vw, 3.5rem) | 700 | -0.02em |
| H3 (карточки) | 1.3rem | 600 | 0 |
| Лейбл | 0.7rem | 500 | 0.25em |
| Основной текст | 1rem–1.1rem | 400 | 0 |
| Мелкий текст | 0.85rem | 400 | 0 |

**Лейблы секций** (например «КТО Я», «УСЛУГИ»): uppercase, letter-spacing 0.25em, цвет `--accent`.

### Настроение и атмосфера

- **Тёмный, технологичный, уверенный** — не агрессивный, не кричащий
- **Воздух и пространство** — щедрые отступы (padding секций: 96–120px)
- **Минимум украшений** — каждый элемент несёт смысл
- **Референсы:** Vercel, Linear, Anthropic, Stripe

## 5. АНИМАЦИИ И ИНТЕРАКТИВ

### Обязательные (влияют на восприятие качества)

| Элемент | Анимация | Реализация |
|---------|----------|-----------|
| Hero H1 | Clip-path reveal, слово за словом | CSS keyframes + stagger |
| Scroll reveal | Fade + translateY(40px) при появлении в viewport | IntersectionObserver |
| Кнопки :hover | translateY(-3px) + glow усиливается | CSS transition |
| Кнопка primary | Пульсирующий box-shadow (pulse) | CSS animation infinite |
| Карточки :hover | translateY(-6px) + border-color accent | CSS transition |
| FAQ accordion | Плавное раскрытие max-height | CSS transition |
| Прогресс скролла | Полоска сверху страницы | CSS scroll-driven API |
| Форма input | border-bottom анимация при фокусе | CSS ::after transition |
| Floating кнопка | Появляется после 500px скролла | JS + opacity transition |

### Hero фон (canvas)

```
- 60 частиц, движутся с скоростью 0.2–0.4px/кадр
- Соединительные линии между частицами < 130px
- Opacity линий: (1 - distance/130) * 0.12
- Цвет частиц: rgba(99, 102, 241, 0.4)
- Gradient orb: 500px, blur 80px, pulsating scale 1→1.15
- Noise texture: opacity 0.03, анимация grain (translate смещения)
```

### Кастомный курсор (только десктоп)

```
- Маленькая точка 8px: следует мгновенно
- Кольцо 36px: следует с lerp 0.12 (инерция)
- mix-blend-mode: difference на точке
- При hover на кнопке: кольцо увеличивается до 56px
- На мобиле: не показывается
```

### Micro-interactions

```
- Навигация: font-weight 400 → 600 при hover (плавно)
- CTA-кнопка: физическое нажатие translateY(3px) при :active
- Счётчик «Открыт для проектов»: точка пульсирует каждые 2 сек
- Теги/бейджи: появляются stagger при входе в viewport
```

## 8. ПРОИЗВОДИТЕЛЬНОСТЬ

### Требования

| Метрика | Цель |
|---------|------|
| LCP (загрузка главного контента) | < 2.5 сек |
| FID / INP | < 100ms |
| CLS | < 0.1 |
| Общий вес страницы | < 500KB |
| Вес без шрифтов | < 200KB |

### Оптимизация

- Шрифт Inter: только latin + latin-ext, только нужные weights (400, 600, 700, 800)
- Canvas: `requestAnimationFrame` с `cancelAnimationFrame` при уходе со страницы
- Изображения: WebP, `loading="lazy"` для всего кроме hero
- CSS: критический CSS в `<head>`, остальное — в конце `<body>`
- JS: весь в одном `<script>` в конце, никаких внешних библиотек
