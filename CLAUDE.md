# Правила работы с проектом SULTANOVA AI

## Структура файлов

- Все страницы — в одном HTML-файле (`index.html`)
- Никаких внешних библиотек, фреймворков и CDN
- CSS — только внутри `<style>` в `<head>`
- JS — только внутри `<script>` в конце `<body>`
- Иконки — SVG inline, не иконочные шрифты
- Шрифты — системный стек (`Inter, Segoe UI, system-ui`)

## Дизайн-система

### Цвета (CSS-переменные)

```css
--bg:        #0A0A0F   /* основной фон */
--bg2:       #0D1117   /* фон карточек */
--bg3:       #111827   /* вложенные элементы */
--p:         #4f46e5   /* акцент индиго */
--ph:        #6366f1   /* hover индиго */
--a:         #06b6d4   /* акцент cyan */
--ah:        #22d3ee   /* hover cyan */
--txt:       #F0F4FF   /* основной текст */
--muted:     #8892A4   /* второстепенный текст */
--border:    rgba(255,255,255,0.07)
--grad:      linear-gradient(135deg,#4f46e5,#06b6d4)
--grad-text: linear-gradient(135deg,#fff 20%,#c7d2fe 55%,#22d3ee 100%)
--radius:    16px
--radius-sm: 10px
```

### Типографика

- Заголовки секций: `clamp(1.9rem, 5vw, 3rem)`, weight 800, letter-spacing -0.03em
- Hero H1: `clamp(1.85rem, 4.5vw, 3.6rem)`, weight 800, line-height 1.12
- Подзаголовки карточек: 1rem – 1.2rem, weight 700
- Основной текст: 0.9rem – 1.05rem, line-height 1.65 – 1.9
- Мелкий текст / метки: 0.68rem – 0.82rem
- Лейблы секций (`.lbl`): 0.68rem, uppercase, letter-spacing 0.28em, цвет `--a`

### UI-компоненты

- Карточки: `background: var(--bg2)`, `border: 1px solid var(--border)`, `border-radius: var(--radius)`
- Hover карточек: `border-color` светлее + `transform: translateY(-4px)`
- Кнопки: `border-radius: 100px`, два варианта — `.btn-p` (градиент) и `.btn-s` (outline)
- Теги / пилюли: `border-radius: 100px`, фон с opacity от акцентного цвета
- Секции чередуются: обычный фон → `.sec-dark` (bg2) → `.sec-grad` (с тонким градиентом)

### Эффекты

- Glassmorphism на навбаре: `backdrop-filter: blur(20px)`
- Glow на hover: `box-shadow: 0 0 40px rgba(79,70,229,.35)`
- Анимация появления при скролле: класс `.rv` + `.on` через IntersectionObserver
- Задержки анимации: `.d1` – `.d5` (0.1s – 0.5s)
- Градиентный текст: `background-clip: text`, `-webkit-text-fill-color: transparent`

## Адаптивность

- Подход: mobile-first
- Брейкпоинты: `480px` / `768px` / `900px`
- Сетка услуг: 4 колонки → 2 → 1
- Bento-сетка: 4 колонки → 2 → 1
- На мобильном: уменьшать padding секций, скрывать декоративные элементы
- Курсор и частицы canvas скрыты на мобильных (`@media max-width: 768px`)
- Кнопки на мобильном растягиваются на всю ширину

## Стиль текста

- Язык: русский, обращение на «вы» (к клиенту)
- Тон: уверенный, прямой, без воды и корпоративщины
- Без слов: «уникальный», «профессиональный», «качественный», «инновационный»
- Заголовки — короткие, конкретные, часто с глаголом действия
- Описания — что получает клиент, а не что я умею делать
- Не «я делаю сайты», а «вы получаете инструмент, который работает»
- Цифры и сроки — только реальные

## Перед любым изменением

Перед тем как создать новый файл, добавить секцию или сделать крупное изменение —
**сначала написать, что планируется сделать**, и дождаться подтверждения.

Мелкие правки (текст, цвета, отступы) — можно делать сразу.

## Коммиты

- Делать только по явной просьбе
- Сообщение на русском, кратко описывает суть изменений
